# Weather-Dashboard


### This is a weather application that allows users to check weather in any city for the next five consecutive days.
#### It's built with 
* HTML
* CSS
* Javacode
* use of openweathermap API
#### Powered by 
* Bootstrap
* jQuery
* Moment.js
* Popper
* Fontawesome
